Records Series API

This is Records Series API
Explain how to use on this VOD in Records Series category.
You can use on 3 methods in this category:
showAllRecords,findSeriesByName and getSeries

Present your series records:

http verb:get

/showAllRecords

If you would like to see all your series records you will have to go into this link:
https://vodrecords.herokuapp.com/showAllRecords
You will get a response in json view

Looking for series by series's name:

http verb:post

/findSeriesByName

If you would like to looking for series by series's name you will have to go into this link:
https://vodrecords.herokuapp.com/findSeriesByName
Then send name with value in the body.
If there is a series with this name, you will get a response in json view with this series
message error in json view otherwise

example of error message

{
 "error": "series not found by name"
}

Looking for series by series's channel and category:

http verb:post

/getSeries

If you would like to looking for series by series's channel and series category you will have to go into this link:
https://vodrecords.herokuapp.com/getSeries
Then send channel with value and category with value in the body.
If there is a series with this channel and this category, you will get a response in json view with all of series with this channel and this category
message error in json view otherwise
